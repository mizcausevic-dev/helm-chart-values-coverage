# Security Policy

`helm-chart-values-coverage` is a pure-transform library and CLI: it reads YAML / `.tpl` files from a chart directory and emits a structured findings report. No network listener, no remote fetch, no Helm CLI invocation, no execution of user-supplied templates.

## Supported versions

Only the latest tagged release is supported.

## Reporting a vulnerability

Please use GitHub Security Advisories for private disclosure:

- [Open a security advisory](https://github.com/mizcausevic-dev/helm-chart-values-coverage/security/advisories/new)

Do not file public issues for security reports.
